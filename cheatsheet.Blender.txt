Definitions
    bevel -- split an edge into 2 or move spaced edges
    edge loop -- connected edges around a 3D object
    edge ring -- a set of edges around a 3D object
    edge slide -- move an edge ring along a surface without deforming the shape
    gizmos -- on-screen tools
    icosphere -- sphere containing only polygon surfaces
    inset face -- small face inside an bigger face
    loop cut -- add a set of edges around the surface of a 3D object
    non-manifold objects -- objects with intersecting faces that are unprintable or unbuildable
    n-gon -- a face with more than 4 sides, which cause problems blocking edge loops and smoothing
    Orthographic -- without perspective in 3D
    uv sphere -- sphere with longitude and latitude lines and containing triangular faces
    vertex slide -- move a vertex along the surface of a 3D object

Acronyms
    HDRI -- High Dynamic Range Image -- adds luminosity to rendered images

3D Modelling Tools
    Extrude -- for selected faces, use <ctrl> to extrude in increments, shortcut e to extrude. <alt> e for extrude menu
    Subdivide -- select a face or edge and right click > Subdivide
    Loop Cut -- create a set of edges surrounding a 3D object, or <ctrl> r, and click and drag to move the loop cut
    Offset Edge Loop Cut -- select an edge loop, choose loop cut tool, subselect offset edge loop cut, click and drag the edge loop
    Selecting: Edge Loops -- choose tweak selector and <alt> click
    Selecting edge rings -- choose tweak selector and <ctrl> <alt> click
    Edge Slide -- choose the edge slide tool, or select edge loop and type 'gg'
    Vertex Slide -- choose the edge slike tool, sub-select vertex slide, or select vertex and type 'gg'
    Bevel -- choose face or edge, then select bevel tool and drag, or <ctrl> b
    Bevel vertex -- choose vertex, choose bevel tool, drag bevel widget then change affect setting to vertices, or <ctrl> <shft> b
    Inset Faces  -- make a small face inside a face, choose Inset Faces tool
    Normalize scale for beveling and insetting -- Object mode > Object > Apply > Scale (or Rotation and Scale)
