Unreal Engine
    Navigations
        Focus on object -- <f>
        Game View - >g>
        Toggle docking windows -- <F10>
        Move In or Out -- LMB Drag Up or Down
        Walk up -- RMB E
        Walk down -- R<B Q
        Rotate Up or Down-- RMB Drag Up or Down
        Walk left -- RMB A
        Walk right -- RMB D
        Walk in -- RMB W
        Walk out -- RMB S
        Open a new C++ Class -- Menu > Tools > New C++ Class
        Open a new level -- Menu > File > New Level
        Open a new Project -- Menu > File > New Project
        Restore editor layout -- Menu > Window > Load Layout > Default Editor Layout
        Show Details panel -- Menu > Window > Details
        Show Full Screen -- <F11>
        Toggle the Content Drawer -- <Ctrl><Space>
        Toggle World Settings -- Menu > Window > World Settings
        Zoom In or Out -- LMB Drag Up or Down
